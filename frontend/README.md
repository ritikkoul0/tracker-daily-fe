# Daily Tracker Frontend

A React-based frontend for tracking daily activities including learning, sleep, and office hours.

## Features

- 📝 Add daily activities with learning, sleep, and office hours
- 📋 View all recorded activities
- 📊 Visualize data with interactive charts
- 📈 Weekly, monthly, and yearly analysis
- 🎨 Beautiful, responsive UI

## Setup

1. Install dependencies:
```bash
cd frontend
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Prerequisites

Make sure the backend server is running on `http://localhost:8080` before starting the frontend.

## Technologies Used

- React 18
- Axios for API calls
- Recharts for data visualization
- CSS3 for styling

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Runs tests
